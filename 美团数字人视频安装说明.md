# 美团数字人视频安装说明

## 1. 克隆本仓库及官方源码

```bash
git clone --recurse-submodules https://github.com/alicechendejun-pixel/-.git
cd -/美团数字人视频
```

已经克隆过但没有源码目录时：

```bash
git submodule update --init --recursive
```

## 2. 创建环境

```bash
conda create -n longcat-video python=3.10 -y
conda activate longcat-video
pip install torch==2.6.0+cu124 torchvision==0.21.0+cu124 torchaudio==2.6.0 --index-url https://download.pytorch.org/whl/cu124
pip install ninja psutil packaging
pip install flash_attn==2.7.4.post1
pip install -r requirements.txt
conda install -c conda-forge librosa ffmpeg -y
pip install -r requirements_avatar.txt
```

## 3. 从 Hugging Face 下载数字人权重

在 `美团数字人视频` 目录执行：

```bash
pip install "huggingface_hub[cli]"
huggingface-cli download meituan-longcat/LongCat-Video-Avatar-1.5 --local-dir ./weights/LongCat-Video-Avatar-1.5
```

Windows PowerShell 也可以在仓库根目录运行：

```powershell
powershell -ExecutionPolicy Bypass -File .\下载美团数字人视频权重.ps1
```

## 4. 运行数字人视频

请进入 `美团数字人视频` 目录，按照官方 README 中 `LongCat-Video-Avatar-1.5` 的命令运行。使用 1.5 模型时需要加入：

```text
--model_type avatar-v1.5 --use_distill
```

权重目录：

```text
./weights/LongCat-Video-Avatar-1.5
```

## 说明

- 源码通过 Git 子模块引用官方仓库，便于后续同步更新。
- 模型权重体积较大，不上传 GitHub。
- 官方项目采用 MIT License，实际使用仍需遵守官方仓库和模型页面条款。
